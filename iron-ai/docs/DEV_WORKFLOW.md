# Dev Workflow

Install dependencies
- `npm install`

Run the app
- `npm run dev`

Lint the codebase
- `npm run lint`

Format the codebase
- `npm run format`

Run all checks (lint + format check)
- `npm run check`

Pre-commit hook setup
- `npm run prepare`

Notes
- The pre-commit hook runs `npm run lint` and `npm run format`.
